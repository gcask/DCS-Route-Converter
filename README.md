# DCS-Route-Converter
Converts between RouteTool and DTC formats.

## Install

Download the [latest release](./releases/latest), extract it in `Saved Games/DCS/`.

## Usage

From `Options > SPECIAL > Route Tool/DTC Converter`, look up a preset from a theatre, and select which airframe you want to export.
Hitting `Convert` will create a matching DTC file in `Saved Games/DCS/DTC/<airframe>-<theatre>-<preset>.dtc`.

This can be done while in-game.
